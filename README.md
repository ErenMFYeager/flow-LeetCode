# LeetCode Search for Flow Launcher

Search and open LeetCode problems instantly from Flow Launcher.

## Usage
* **`lc two sum`** → Opens the problem page based on the name
* **`lc #1`** or **`lc 1`** → Opens the problem with that specific LeetCode number
* **`lc daily`** → Fetches and opens today's LeetCode Daily Challenge
* **`lc random`** → Fetches and opens a random LeetCode problem

## Status
🚧 In development — MVP, daily challenges, and random problem generation are working. More features coming (solved status, topic filters).
